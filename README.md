# cohort_retention_analysis
User Retention Cohort Analysis

Overview

This project analyzes user retention using cohort analysis techniques. The objective was to evaluate how long users remain active after registration and compare retention patterns between promotional and organic acquisition channels.

Tools Used
PostgreSQL
SQL
Google Sheets
Cohort Analysis
Retention Analysis

Dataset

The project uses two source tables:

cohort_users_raw
cohort_events_raw

The data required extensive preprocessing due to inconsistent date formats and noisy records.

Methodology
Cleaned and standardized date fields.
Joined user and event datasets.
Created monthly cohorts based on registration dates.
Calculated month offsets between registration and activity.
Aggregated active users by cohort and month.
Built retention tables and heatmaps in Google Sheets.
